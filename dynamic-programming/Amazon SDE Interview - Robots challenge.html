Amazon SDE Interview - Robot Position Time Series Challenge

I made this challenge to train a solution for this problem, as I couldn't find the exact same problem anywhere.

Challenge

Given a grid of robot positions over time, determine if the sequence is valid.

Each row represents one time step. Each index can be:

  • 1 = robot present
  • 0 = empty

Rules:

  • The number of robots stays constant.
  • Each robot may move at most 1 index left or right per step.
  • No robot can teleport.

Return true if valid, otherwise false.

Example valid:

[[1,0,0,1],
 [0,1,1,0]]

Example invalid:

[[1,0,0,0,1],
 [1,0,1,0,0]]
